- ✅ 所有样式值从 `designSystem` 引用

## ❓ 常见问题

### Q: 从哪里开始？
A: 如果是新项目，从 [快速开始](./QUICK-START.md) 开始。如果是迁移项目，从 [迁移指南](./MIGRATION-GUIDE.md) 开始。

### Q: 文档太长，如何快速找到我需要的内容？
A: 使用浏览器的搜索功能（Cmd/Ctrl + F），或者查看本文档的快速导航部分。

### Q: 遇到问题怎么办？
A:
1. 先搜索文档中是否有相关说明
2. 查看源码和注释
3. 查看 Ant Design 官方文档
4. 提 Issue 或询问团队

### Q: 如何贡献文档？
A: 欢迎提 PR！请保持文档格式一致，并确保内容准确。
